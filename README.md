# Web Design Project  Website

This project is a complete **five-page website** built with **HTML and CSS**, including images, organized folders, and wireframes to show professional planning and development workflow.

---

##  Project Overview
This project demonstrates essential front-end skills:

- Semantic HTML5  
- Clean and reusable CSS  
- Organized folder structure  
- Image asset management  
- Internal page linking  
- Wireframe planning  

---

## Project Structure

---

## Pages Included

### ✔ Home Page (`index.html`)
- Introduction  
- Hero section  
- Navigation  

### ✔ About Page (`about.html`)
- Description of the site  
- Purpose & mission  

### ✔ Players Page (`players.html`)
- Profiles of 5 players  
- Player images  

### ✔ Gallery / Extra Page (`gallery.html`)
- Extra images  
- Events or highlights  

### ✔ Contact Page (`contact.html`)
- Contact form  
- Social details
